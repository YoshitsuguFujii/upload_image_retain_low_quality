# UploadImageRetainLowQuality
upload image to s3 bucket retain low quality image file

## Installation

install

    $ gem install upload_image_retain_low_quality

## Usage

just do this

    $ upload_image_retain_low_quality

help

    $ upload_image_retain_low_quality --help
